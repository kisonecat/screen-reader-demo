Absolute value

\(\newcommand{\abs}[1]{\sro{absolute value}\left|#1\right|}\) So \(2x\) is twice the value of \(x\).

So \(\abs{x}\) is the absolute value of \(x\).

Cases

\(\newenvironment{srocases}{\sro{cases}\begin{cases}}{\end{cases}}\) \[\abs{x} = \begin{srocases} x & \mbox{if $x \geq 0$} \\ -x & \mbox{if $x < 0$} \end{srocases}\]
